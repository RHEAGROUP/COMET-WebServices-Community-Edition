COMET-WebServices Github page: COMET Webservices

The COMET Webservices is the RHEA Group Concurrent Design REST API. COMET is the RHEA Group Concurrent Design Solution that allows a team of engineers to perform Concurrent Design. COMET is an implementation of ECSS-E-TM-10-25A Annex A and C. ECSS-E-TM-10-25A Annex A is the so-called master model that is expressed in UML. ECSS-E-TM-10-25A Annex C describes the REST API.

COMET-IME Desktop App Github page: COMET IME

The COMET Integrated Modelling Environment (IME) Community Editition (CE) is the RHEA Group open source Concurrent Design desktop application and Microsoft Excel integration compliant with ECSS-E-TM-10-25 Annex A and Annex C. COMET IME makes use of a plugin framework, the end-user functionality is implemented in COMET IME plugins that are loaded at run-time. This makes it straightforward to package, distribute and update the environment.
